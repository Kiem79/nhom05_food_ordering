## 2. Mô tả dự án
**Chủ đề:** Nền tảng đặt món ăn trực tuyến theo nhóm.  
**Tên thương hiệu:** **Foodie - Smart Ordering**
### Tính năng nổi bật
* **Đặt hàng nhóm thông minh:** Cho phép các thành viên kết nối và gộp đơn dễ dàng trên cùng một nền tảng.
* **Chia bill tự động:** Hệ thống tính toán chi phí chính xác cho từng người chỉ trong vài giây, loại bỏ sai sót thủ công.
* **Cinematic UI:** Ngôn ngữ thiết kế hiện đại, mang lại trải nghiệm thị giác thú vị và chuyên nghiệp.

## 4. Hướng dẫn chạy Local
Để khởi chạy dự án trên máy cá nhân, vui lòng thực hiện các bước sau:

1.  **Cài đặt thư viện:**
    ```bash
    npm install
    ```

2.  **Chạy dự án ở chế độ phát triển:**
    ```bash
    npm run dev
    ```
